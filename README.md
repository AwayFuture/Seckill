# Seckill
高并发秒杀系统

## 1. 系统框架
  (1) 前端: Boostrap + JS + JQuery
  
  (2) 后端: SpringMVC + Spring + MyBatis + Redis
  
  (3) DB: MySQL
  
  
## 2. 开发环境
  (1) 开发工具: IntellIJ IDEA 2019
  
  (2) 运行环境: Tomcat 9
  
  (3) 浏览器: 兼容
